Project: TEXT GENERATOR

Primary Tools used: Python (Tkinter); SQLite


Key details:

Cagayan de Oro has many areas that highly prone to flooding, putting residents and responders at risk.

Problem:

Weather Watch on-duty currently making a manual draft of the street situation (i.e. what areas are not passable, what areas that still passable, etc). So it takes time to disseminate the information into social media.

Solution:

Developed “Street Situation Text Generator app” to provide quick updates (10x than manual encoding) on flooded streets, disseminate the latest information regarding street situation via Facebook Page post.

Key details:

One of the job description as Weather Watch staff in City Disaster Risk Reduction and Management Department (CDRRMD) Cagayan de Oro is to make a local post regarding on the Weather updates released by the official facebook page of "Mindanao PAGASA Regional Services Division" (MPRSD); one of the regional offices of PAGASA. When MPRSD released weather updates post; Weather Watch on-duty makes draft on it to make it localized. Localized means Staff on-duty shall just fetch those areas that connected to our Downstream Rivers here in Cagayan de Oro City. Those areas are the Municipalities of Bukidnon (Libona, Talakag, Baungon Dahilayan, Manolo).

Problem:

The method when weather Watch on-duty extracting those mentioned areas are too slow; time consuming and too obsolete, since MPRSD released other areas too in whole Mindanao.

Solution:

Developed a “Keyword Extracting app” to provide quick and much less hassle, to fetch the desired areas on the MPRSD weather updates post. Weather Watch on-duty can now disseminate the latest information up to a minute, regarding weather updates in Cagayan de Oro and its Tributaries.

Project:WEB APPLICATION

Primary Tools used: Python (Flask); MySQL; HTMLS/CSS; Android ADB; Tomorrow.io API (for testing)


Key details:

When disseminating important information, posting on social media is not enough since some areas or barangays in Cagayan de Oro are not always active on social media. This is why it is important to have an alternative: SMS Textblast.

Problem:

One of the agencies in Cagayan de Oro has already developed such an application, but there are hiccups. When sending to the appropriate recipients, it takes a while— to receive a bulk SMS to 100 to 150 recipients takes too much time. Additionally, the CDRRMD does not have access when there is wrong on the app.

Solution:

Build a dedicated SMS Textblast via web app as an alternative way of disseminating information. This time, no API is involved—fully customized using an old Android device, with the ability to via ADB protocol. Personnel can now also check the message history and troubleshoot if necessary.

Key details:

With today's technology, weather forecasting is possible from virtually anywhere in the world. In Cagayan de Oro, CDRRMD has existing partnerships with agencies for monitoring weather-related events. However, this project which is Local Barangay Forecasting serves more as a situational initiative aimed at strengthening local forecasting capabilities.

Problem:

While not a critical issue, there is a gap in localized weather visuals. Currently, CDRRMD relies mainly on ARGs (Automatic Rain Gauges) and WLMS (Water Level Monitoring Systems), which can only monitor the actual status of rainfall and water levels, and are also limited in coverage—especially in barangays near the river.

Solution:

A localized system is being developed using a weather forecasting technology with the help of API to provide forecasts specific to each barangay in Cagayan de Oro. Although still in progress, this system aims to enhance situational awareness on the ground. Current development is affected by API limitations and access restrictions, but once resolved, it will greatly improve the city's capacity to deliver hyperlocal weather updates.
